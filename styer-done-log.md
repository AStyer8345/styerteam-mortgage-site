# Styer Mortgage — Done Log

Append-only log of tasks marked RESOLVED or PROGRESS by scheduled tasks and the briefing page.

**Formats:**
- `[YYYY-MM-DD HH:mm] [system] {id} — RESOLVED | {note}`
- `[YYYY-MM-DD HH:mm] [system] {id} — PROGRESS {pct}% | {note}`

RESOLVED moves the task's card into the Done tab on improvement-plan.html.
PROGRESS updates the card's progress bar without moving it.
Later PROGRESS lines overwrite earlier ones (latest wins).
A RESOLVED line supersedes all prior PROGRESS lines for the same id.

Reconciled into HTML by the `styer-apply-done` scheduled task.
Do not edit this file by hand unless backfilling.
