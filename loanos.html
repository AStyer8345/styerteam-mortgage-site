Key Decisions
Supabase = Memory Layer
Automations already fire (contract, CD, pre-approval). Supabase adds the record layer underneath every trigger — contact, loan, document, and activity log persist after each automation runs.
n8n Replaces Zapier
Self-hosted n8n at ~$15/mo vs Zapier at $100-300/mo at automation volume. Native Supabase node + Claude API node = fewer moving parts, more control, no per-task billing.
Mortgage Coach Replacement
6 calculators replace Mortgage Coach entirely. Differentiator: Claude writes plain-English narrative summary per scenario. Output is branded PDF or shareable link integrated with loan records.
Build For Yourself First
Phase 1-2 = personal production tool replacing Jungo + Mortgage Coach. Phase 4 = license to LOs nationwide at $97-397/mo after personal build is proven and stable.
